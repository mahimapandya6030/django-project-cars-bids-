# AutoMax 🚗

A web application to buy and sell cars with features like listing, liking, and booking test drives.

---

##  Features
- User authentication (Login / Signup)
- Create, edit, delete listings
- Like / Unlike listings ❤️
- Book test drives
- Pagination and filters

---

##  Tech Stack
- Python
- Django
- HTML, CSS, Bootstrap
- JavaScript / jQuery
